# Page Structure

Here is a brief overview of all the pages included in this template:

- `index.html` (or `Home 1`) — The primary landing page focusing on an immersive first impression.
- `pages/home-2.html` — An alternative homepage layout with different header designs.
- `pages/portfolio.html` — Grid layout for exploring all custom furniture case studies.
- `pages/portfolio-single.html` — Deep-dive case study page for a specific project.
- `pages/about.html` — The story behind the studio and the artisan. 
- `pages/services.html` — Services outline and pricing tiers.
- `pages/process.html` — The 6-step woodworking project timeline timeline.
- `pages/materials.html` — An educational page showcasing wood species and finishing choices.
- `pages/shop.html` — Grid overview of available physical products for instant purchase.
- `pages/shop-single.html` — Details page for a specific wooden object or furniture piece.
- `pages/quote.html` — Dynamic form for requesting custom furniture commissions.
- `pages/contact.html` — Standard contact information and general inquiry form.
- `pages/faq.html` — Frequently asked questions regarding ordering, shipping, and care.
