# Installation Guide

Setting up this template is straightforward as it consists of static HTML, CSS, and vanilla JavaScript. Tailwind CSS is used via a CDN for ease of deployment.

**Step-by-step Setup:**
1. **Extract the Files**: Unzip the downloaded template package.
2. **Review the Structure**: You will see an `assets` folder containing `css`, `images`, and `js`. HTML pages are stored in the root directory and the `pages` directory.
3. **Local Testing**: No build step is required! You can open any HTML file (such as `index.html`) directly in your browser. Alternatively, for a better development experience, you can use a local server like VS Code's "Live Server" extension.
4. **Icons**: The template relies on Iconify (`https://code.iconify.design/3/3.1.1/iconify.min.js`). Ensure you have an active internet connection to load the icons.
